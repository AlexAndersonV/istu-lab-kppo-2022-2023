# Функции системы
1. Загрузка сохранения
* Действия: открывается соответствующий файл сохранения, десериализуется из JSON, инициализируется экран холла казино.
* Вход: id игрока, id сохранения
* Выход: десериализованный экземпляр класса игры
2. Сохранение в файл
* Действия: сериализация прогресса игрока, сохранение в файл
* Вход: id игрока, id сохранения, экземпляр класса игры
* Выход: файл сохранения
3. Показать правила игры 
* Действия: открыть всплывающее окно с информацией о текущей игре
* Вход: id игры
* Выход: окно с соответствующей информацией
4. Начать игру в рулетку
* Действия: инициализировать игру в рулетку
* Вход: -
* Выход: инициализированный экземпляр класса "Рулетка"
5. Сделать ставку в рулетку
* Действия: игрок выбирает ставки
* Вход: пользовательский ввод ставок
* Выход: сектора рулетки и ставки на них
6. Крутить рулетку
* Действия: анимация раскрутки рулетки, генерация случайного числа
* Вход: -
* Выход: выпавшее число
7. Начать игру в блэкджек
* Действия: инициализировать игру в блэкджек, создать колоду
* Вход: -
* Выход: инициализированные экземпляры классов "Блэкджек" и "Колода"
8. Сделать ставку в блэкджек
* Действия: игрок выбирает ставку
* Вход: пользовательский ввод ставки
* Выход: значение ставки
9. Взять карту
* Действия: из конца колоды карта передается игроку/дилеру
* Вход: id получателя
* Выход: добавление карты в руку
10. Не брать карту
* Действия: передать ход дилеру
* Вход: -
* Выход: действия дилера
11. Удвоить ставку
* Действия: Ставка игрока удваивается, передается ход дилеру
* Вход: -
* Выход: действия дилера
12. Начать игру в слоты
* Действия: инициализировать игру в слоты
* Вход: -
* Выход: инициализированный экземпляр класса Слоты
13. Сделать ставку в слоты
* Действия: игрок выбирает ставку
* Вход: пользовательский ввод ставки
* Выход: значение ставки
14. Дернуть рычаг слот-машины
* Действия: анимируются 3 окна слот-машины, выбирается 3 случайных числа
* Вход: ставка
* Выход: выигрыш игрока
15. Начать игру в стаканчики
* Действия: инициализировать игру в стаканчики, выбрать 3 случайных числа, анимировать перемешку стаканчиков
* Вход: -
* Выход: инициализированный экземпляр класса Стаканчики
16. Выбрать стаканчик
* Действия: игрок выбирает один из 3-х стаканчиков и получает соответствующий выигрыш
* Вход: пользовательский выбор стаканчика
* Выход: выигрыш игрока
17. Сдаться
* Действия: вычитание из баланса игрока его ставки, обнуление экземпляра класса текущей игры
* Вход: ставка
* Выход: возврат игрока в холл казино
